# HTML

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Absolute Centering</title>
</head>
<body>
    <div class="container">
        <div class="centered">Centered</div>
    </div>
</body>
</html>
```

# CSS

```css
.container {
    width: 300px;
    height: 300px;
    background-color: #f8f8f8;
    position: relative;
    border: 1px solid #ccc;
    margin: 50px auto;
}

.centered {
    width: 100px;
    height: 50px;
    background-color: #ff6b6b;
    color: white;
    text-align: center;
    line-height: 50px;
    
    position: absolute;
    top: 50%;
    left: 50%;
    transform: translate(-50%, -50%);
}
```
