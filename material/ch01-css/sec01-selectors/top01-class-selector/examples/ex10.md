# HTML

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>CSS Tooltip</title>
</head>
<body>
    <div class="tooltip-container">
        Hover Me
        <div class="tooltip">I am a tooltip!</div>
    </div>
</body>
</html>
```

# CSS

```css
.tooltip-container {
    display: inline-block;
    position: relative;
    margin: 100px;
    padding: 10px 20px;
    background-color: #333;
    color: white;
    cursor: pointer;
    border-radius: 4px;
}

.tooltip {
    visibility: hidden;
    width: 120px;
    background-color: black;
    color: #fff;
    text-align: center;
    border-radius: 6px;
    padding: 5px 0;
    
    /* Position the tooltip */
    position: absolute;
    bottom: 120%; /* Above the container */
    left: 50%;
    transform: translateX(-50%); /* Center horizontally */
    
    /* Fade in effect */
    opacity: 0;
    transition: opacity 0.3s;
}

/* Arrow */
.tooltip::after {
    content: "";
    position: absolute;
    top: 100%; /* At the bottom of the tooltip */
    left: 50%;
    margin-left: -5px;
    border-width: 5px;
    border-style: solid;
    border-color: black transparent transparent transparent;
}

.tooltip-container:hover .tooltip {
    visibility: visible;
    opacity: 1;
}
```
