# HTML

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Absolute Position Basic</title>
</head>
<body>
    <div class="box">Normal Box</div>
    <div class="absolute-box">Absolute Box</div>
    <div class="box">Normal Box</div>
</body>
</html>
```

# CSS

```css
body {
    background-color: #f0f0f0;
    padding: 20px;
}

.box {
    width: 100px;
    height: 100px;
    background-color: #ccc;
    margin-bottom: 10px;
}

.absolute-box {
    width: 100px;
    height: 100px;
    background-color: coral;
    position: absolute;
    top: 50px;
    right: 50px;
}
```
