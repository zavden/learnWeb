# HTML

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Absolute inside Relative</title>
</head>
<body>
    <div class="container">
        <div class="child">I am absolute</div>
    </div>
</body>
</html>
```

# CSS

```css
.container {
    width: 300px;
    height: 300px;
    background-color: #eee;
    border: 2px solid #333;
    position: relative; /* Context for absolute child */
    margin: 50px;
}

.child {
    width: 100px;
    height: 100px;
    background-color: lightgreen;
    position: absolute;
    bottom: 20px;
    right: 20px;
}
```

# JavaScript

```javascript
console.log('The child is positioned relative to the .container because the container has position: relative.');
```
