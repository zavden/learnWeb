# HTML

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Image Caption Overlay</title>
</head>
<body>
    <figure class="figure">
        <img src="https://via.placeholder.com/400x300" alt="Placeholder Image">
        <figcaption class="caption">Beautiful View</figcaption>
    </figure>
</body>
</html>
```

# CSS

```css
.figure {
    position: relative;
    width: 400px; /* Match image width */
    margin: 20px;
    padding: 0;
}

img {
    display: block; /* Remove bottom space */
    width: 100%;
}

.caption {
    position: absolute;
    bottom: 0;
    left: 0;
    width: 100%;
    background-color: rgba(0, 0, 0, 0.6);
    color: white;
    padding: 10px;
    box-sizing: border-box; /* Include padding in width */
    text-align: center;
}
```
