# HTML

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Viewport vs ViewBox</title>
</head>
<body>
    <h2>Viewport (300x200) vs ViewBox (0 0 100 100)</h2>
    <!-- The SVG canvas is 300x200 pixels -->
    <!-- The internal coordinate system is 0-100 units wide and tall -->
    <svg width="300" height="200" viewBox="0 0 100 100" class="svg-container">
        <!-- A rectangle from 0,0 to 100,100 fills the entire viewbox -->
        <rect x="0" y="0" width="100" height="100" fill="#4caf50" stroke="#333" stroke-width="2" />
        <circle cx="50" cy="50" r="20" fill="#fff" />
    </svg>
</body>
</html>
```

# CSS

# JavaScript

```javascript
console.log('The viewBox "0 0 100 100" maps 100 user units to the 300px width and 200px height of the viewport.');
```
