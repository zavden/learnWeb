# HTML

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Rotation Center</title>
</head>
<body>
    <svg width="400" height="300" class="svg-container">
        <!-- Center point for reference -->
        <circle cx="200" cy="150" r="5" fill="black" />

        <!-- Rotated around origin (0,0) by default - usually flies off screen if not careful -->
        <!-- We use rotate(angle, cx, cy) to rotate around a specific point -->
        
        <!-- Normal -->
        <rect x="150" y="100" width="100" height="100" fill="rgba(255,0,0,0.3)" stroke="red" />

        <!-- Rotated 45 degrees around center (200, 150) -->
        <rect x="150" y="100" width="100" height="100" fill="rgba(0,0,255,0.3)" stroke="blue"
              transform="rotate(45, 200, 150)" />
    </svg>
</body>
</html>
```

# CSS
