# HTML

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Line Coordinates</title>
</head>
<body>
    <svg width="300" height="200" class="svg-container">
        <!-- Start point (x1, y1) -> End point (x2, y2) -->
        <line x1="50" y1="50" x2="250" y2="150" stroke="purple" stroke-width="5" />
        
        <!-- Markers for points -->
        <circle cx="50" cy="50" r="5" fill="red" />
        <text x="50" y="40">x1, y1</text>
        
        <circle cx="250" cy="150" r="5" fill="blue" />
        <text x="250" y="170">x2, y2</text>
    </svg>
</body>
</html>
```
