# HTML

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>User Units and Scaling</title>
</head>
<body>
    <svg width="400" height="200" viewBox="0 0 400 200" class="svg-container">
        <!-- Normal Scale (1:1) -->
        <rect x="10" y="10" width="50" height="50" fill="blue" />
        <text x="10" y="80" font-size="12">Scale 1</text>

        <!-- Scaled Group (2x) -->
        <g transform="translate(100, 0) scale(2)">
            <!-- This rect will appear 100x100px -->
            <!-- Coordinates are also scaled! x=10 becomes x=20 on screen relative to group origin -->
            <rect x="10" y="10" width="50" height="50" fill="green" opacity="0.7" />
            <text x="10" y="40" font-size="6" fill="white">Scale 2</text>
        </g>
    </svg>
</body>
</html>
```
